### Python commands

create a virtual environment
```bash
python3 -m venv env
```

activate the virtual environment
```bash
source env/bin/activate
```

deactivate the virtual environment
```bash
deactivate
```

install dependencies
```bash
pip install -r requirements.txt
```

run uviorn
```bash
uvicorn main:app --reload
```
